# 💬 Real-Time Chat Application

A real-time chat app built using **Node.js**, **Express**, and **Socket.IO**. Users can connect and chat live in a shared chatroom through WebSockets.

---

## 🚀 Features

- Real-time messaging with WebSockets
- Socket.IO for smooth bidirectional communication
- Simple and clean user interface
- Auto-updating message feed

---

## 📁 Project Structure

```
realtime_chat/

│   └── style.css       # Styling

```

---

## 🖥️ How to Run Locally

1. Clone this repository or download the ZIP
2. Open terminal and navigate to the project folder
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the server:
   ```bash
   node server.js
   ```
5. Open your browser and go to:
   ```
   http://localhost:3000
   ```
